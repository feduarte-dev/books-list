## How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/feduarte-dev/books-list
   ```

2. Navigate to the project directory:

   ```bash
   cd your-repository
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Test the application:

   ```bash
   npm run test
   ```
